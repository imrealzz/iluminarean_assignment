# Project

이 프로젝트는 일루미나리안 무료 체험 신청 동작에 대한 자동화 시나리오 입니다.

## 설치

해당 프로젝트는 python 및 selenium 을 사용했습니다.

```bash
brew install python3
pip3 install selenium
```

- Chrome Driver 링크 주소 : https://chromedriver.chromium.org/downloads
